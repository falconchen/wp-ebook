A simple plugin, which will allow us to add multiple ebook from the Wordpress admin panel. User can download the ebook without logged into the system.

== Description ==
Ebook Downloader is a simple plugin. After installing the plugin site administrator can upload the ebooks into the system. Front end user can download those ebooks by making payemnt, but they donot need to logged into the system.

= Add Ebooks =
To add a ebook you need to
 * Enter ebook Title;
 * Enter Description;
 * Enter Price for the ebook;
 * Upload the ebook;
 
= Update and Delete ebook =
 It will be very easy to update and delete any ebook, just go to the ebook list to update / delete it.
 
= Shortcode =
[ebook_downloader]

== Installation ==
1.Download the Ebook downloader zip file and extract to the  `/wp-content/plugins/` directory.
2.Activate the plugin through the 'Plugins' menu in WordPress Admin.

== Frequently Asked Questions ==
= Where I will find the ebook option? =
You will find the Ebook in the left memu bar of the Wordpress Administration panel

= How ebooks will be visible from the website front end? =
After adding the ebooks, you need to add this shortcode '[ebook_downloader]' into a page or a section, where your visitor will download those ebooks. 

= What is the Shortcode? =
[ebook_downloader]

= What is 'Paypal Setting'? =
Here you need to add your Paypal Live Merchant ID.

= Should I need to make payment to download the ebook? =
Yes
